
# 产品功能总结

## 一、浏览器插件 (xhs_helper_browser_extension)

### 核心功能

#### 1. 提示词管理
- 自定义提示词的添加、编辑、删除、搜索
- 一键插入提示词到AI对话输入框
- 支持多平台：豆包、文心一言、DeepSeek
- 提示词云端同步（Chrome Sync）

#### 2. 小红书搜索优化
- **高价值笔记筛选**：自动筛选点赞数超过阈值的笔记
- **关键词拓展**：基于下拉提示词，批量拓展相关关键词
- 支持多种排序方式：最多点赞、最多收藏、最多评论、最新发布
- 支持发布时间过滤：一天内、一周内、一个月内、一年内

#### 3. 小红书图片下载
- 自动在小红书页面显示下载按钮
- 支持批量下载笔记中的所有图片
- 自动注入下载按钮到发现页和详情页

#### 4. 激活码认证系统
- 支持激活码认证，解锁高级功能
- 每日使用次数限制
- 设备码绑定机制
- 支持永久有效和限时激活码

### 支持的网站
- 小红书：搜索优化、图片下载
- 豆包、文心一言、DeepSeek：提示词插入

---

## 二、PC 客户端 (xhs_helper_pc_server)

### 核心功能

#### 1. 多设备管理
- 支持同时连接管理多个 Android 设备，独立调度

#### 2. 智能养号流程
- 启动发现 → 随机浏览 → 关键词搜索 → 访问笔记 → 模拟互动，全自动化

#### 3. 关键词管理
- Web 界面增删查改，支持批量导入

#### 4. 行为模拟
- 随机点赞/收藏/评论，模拟真实用户行为

#### 5. 参数化配置
- 可配置时长、比例、概率，灵活调整策略

#### 6. Web 管理界面
- 可视化操作，不用命令行

#### 7. 笔记创建功能
- 支持自动发布笔记

#### 8. 文件传输功能
- 文件夹传文件到手机
- PDF 转图片
- 图片上传百度云生成分享链接

#### 9. 激活码认证系统
- 支持激活码认证，解锁高级功能
- 设备数限制、每日养号次数限制、每日内容制作次数限制、每日导出分享次数限制
- 设备码绑定机制

### 技术栈
- 自动化控制：uiautomator2 + ADB
- Web 框架：Flask
- 部署：本地 PC 服务器部署
- 支持系统：Linux / macOS / Windows

## 四、服务端 API (activation-api)

### 主要功能

#### 1. 客户端接口（供 Chrome 插件、PC 客户端调用）
- 验证激活码接口 (`/auth/verify`)
- 查询设备信息接口 (`/device/info`)

#### 2. 管理端接口（供管理后台调用）
- 生成激活码接口 (`/auth/generate`)
- 查询激活码信息接口 (`/auth/info`)
- 更新激活码信息接口 (`/auth/update`)
- 删除激活码接口 (`/auth/delete`)
- 列出所有激活码接口 (`/auth/list`)
- 设备管理接口
- 权限管理接口

---

## 总结

这两个项目都是为小红书自媒体作者提升创作效率而设计的工具：
- **浏览器插件**：主要在浏览器端提供搜索优化、图片下载和提示词管理功能
- **PC 客户端**：提供自动化养号和笔记创建功能
- 两者都通过激活码系统进行授权管理，采用分级套餐的商业模式
